
## Setting up a virtual environment: 

- windows (python 3.10):
```
py -3.10 -m pip install virtualenv
```
```
py -3.10 -m venv env
```
```
env\Scripts\activate
```

- linux & mac:
```
pip3 install virtualenv
```
```
virtualenv env
```
```
source env/bin/activate
```

## Install the packages
```
pip install flask flask-sqlalchemy
```
